SQL Constraint 限制

Constraint 用來有條件地限制哪些資料才可以被存入資料表中，也就是對欄位作約束。這些限制可以在建立資料表時 CREATE TABLE 指定條件，或是之後再修改 ALTER TABLE 指定。

SQL 有哪幾種類型的限制條件？

  • NOT NULL 非空值限制
  • UNIQUE 唯一限制
  • PRIMARY KEY 主鍵限制
  • FOREIGN KEY 外鍵限制
  • CHECK 檢查限制
  • DEFAULT 預設值限制
