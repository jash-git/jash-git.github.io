LCASE() 函數 (SQL LCASE() Function)

LCASE() 函數用來將英文字母轉換為小寫。

LCASE() 語法 (SQL LCASE() Syntax)

SELECT LCASE(column_name) FROM table_name;

LCASE() 函數用法 (Example)

假設我們想從下面的 customers 資料表中查詢顧客姓名並以小寫返回結果：

C_IdName
1Smith
2Brad

我們可以這樣寫：

SELECT LCASE(Name) FROM customers;

返回的結果如下：

LCASE(Name)
smith
brad
