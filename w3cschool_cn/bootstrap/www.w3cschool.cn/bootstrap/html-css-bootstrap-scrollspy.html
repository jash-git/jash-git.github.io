Bootstrap 滚动监听

由 wcxstyle 创建，youj 最后一次修改 2017-01-05

ScrollSpy广泛用于只有一个页面的网站。

ScrollSpy监听任何DOM元素中的滚动，并根据元素的滚动位置突出显示导航栏中的菜单项。

它由一个导航栏和一个内容区域组成。

内容区域被划分为多个部分，每个部分具有唯一的ID。

导航栏仅包含具有节ID作为其href属性值的内部链接。

要使ScrollSpy正常工作，我们应该向内容部分添加两个自定义数据属性： data-spy和 data-target。

data-spy告诉Bootstrap内容区域是用于ScrollSpy插件的。

data-offset属性告诉Bootstrap在激活ScrollSpy插件之前从顶部离开多少像素。
